Libraries first:


Create the libraries folder in docroot, then place all library folders in this folder. Note that when first unzipped the folder names include the word “master,” remove those from the folder names. 


1) dropzone library - https://github.com/enyo/dropzone


2) imagesLoaded library - https://github.com/desandro/imagesloaded


3) Masonry library -  https://github.com/desandro/masonry


4) Cropper library - https://github.com/fengyuanchen/cropper

Media Modules


  1. Chaos Tools Suite - https://ftp.drupal.org/files/projects/ctools-8.x-3.0.tar.gz
  2. Entity (API) - https://ftp.drupal.org/files/projects/entity-8.x-1.0-alpha4.tar.gz
  3. Embed - https://ftp.drupal.org/files/projects/embed-8.x-1.0.tar.gz
  4. Entity Embed (API) - https://ftp.drupal.org/files/projects/entity_embed-8.x-1.0-beta2.tar.gz (Beta)
  5. Entity Browser - https://ftp.drupal.org/files/projects/entity_browser-8.x-1.0.tar.gz (comes with Entity Browser IEF inline entity form)
  6. Media Entity - https://ftp.drupal.org/files/projects/media_entity-8.x-1.6.tar.gz (the base media module that other media modules can extend from)
  7. Inline Entity Form - https://ftp.drupal.org/files/projects/inline_entity_form-8.x-1.0-beta1.tar.gz (Beta)
  8. Media Entity Image - https://ftp.drupal.org/files/projects/media_entity_image-8.x-1.2.tar.gz (other media entity type modules - Media Entity Instagram, Media Entity Twitter, Media Entity YouTube, etc)
  9. Media Entity Document - https://ftp.drupal.org/files/projects/media_entity_document-8.x-1.1.tar.gz
  10. Crop API - https://ftp.drupal.org/files/projects/crop-8.x-1.2.tar.gz (comes with Media entity crop)
  11. ImageWidgetCrop - https://ftp.drupal.org/files/projects/image_widget_crop-8.x-2.0.tar.gz
  12. DropzoneJS - https://ftp.drupal.org/files/projects/dropzonejs-8.x-1.0-alpha7.tar.gz (Alpha - for Entity browser widget - dependency - Entity Browser)
  13. DropzoneJS Entity Browser
  14. Video Embed Field - https://ftp.drupal.org/files/projects/video_embed_field-8.x-1.5.tar.gz
  15. File Entity Browser - https://ftp.drupal.org/files/projects/file_browser-8.x-1.1.tar.gz (comes with (installs) File Browser?)
  16. STOP - Enable remaining modules that are components of other modules (except samples)
  17. Media Entity Browser - https://ftp.drupal.org/files/projects/media_entity_browser-8.x-1.x-dev.tar.gz (Install the Dev version and install last)
