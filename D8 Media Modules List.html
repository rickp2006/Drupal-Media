Media:


Libraries first:


1) dropzone library - https://github.com/enyo/dropzone

Place it in the libraries folder (/libraries), the full path to the required js file should be: /libraries/dropzone/dist/dropzone.js


2) imagesLoaded library - https://github.com/desandro/imagesloaded releases and place it in /libraries


3) Masonry library -  https://github.com/desandro/masonry and place it in /libraries


4) Cropper - https://github.com/fengyuanchen/cropper and place it in /libraries

Media Modules


  1. Entity (API) - https://ftp.drupal.org/files/projects/entity-8.x-1.0-alpha4.tar.gz
  2. Embed - https://ftp.drupal.org/files/projects/embed-8.x-1.0.tar.gz
  3. Entity Embed (API) - (Alpha) https://ftp.drupal.org/files/projects/entity_embed-8.x-1.0-beta2.tar.gz (Beta)
  4. Media Entity - https://ftp.drupal.org/files/projects/media_entity-8.x-1.6.tar.gz (the base media module that other media modules can extend from)
  5. Inline Entity Form - https://ftp.drupal.org/files/projects/inline_entity_form-8.x-1.0-beta1.tar.gz (Beta)
  6. Entity Browser - https://ftp.drupal.org/files/projects/entity_browser-8.x-1.0.tar.gz (comes with Entity Browser IEF inline entity form)
  7. Media Entity Browser - https://ftp.drupal.org/files/projects/media_entity_browser-8.x-1.x-dev.tar.gz (Install the Dev version)
  8. Media Entity Image - https://ftp.drupal.org/files/projects/media_entity_image-8.x-1.2.tar.gz (other media entity type modules - Media Entity Instagram, Media Entity Twitter, Media Entity YouTube, etc)
  9. Media Entity Document - https://ftp.drupal.org/files/projects/media_entity_document-8.x-1.1.tar.gz
  10. Crop API - https://ftp.drupal.org/files/projects/crop-8.x-1.2.tar.gz (comes with Media entity crop)
  11. ImageWidgetCrop - https://ftp.drupal.org/files/projects/image_widget_crop-8.x-2.0.tar.gz
  12. Chaos Tools Suite - https://ftp.drupal.org/files/projects/ctools-8.x-3.0.tar.gz
  13. File Entity Browser - https://ftp.drupal.org/files/projects/file_browser-8.x-1.1.tar.gz (comes with File Browser?)
  14. DropzoneJS - https://ftp.drupal.org/files/projects/dropzonejs-8.x-1.0-alpha7.tar.gz (Alpha - for Entity browser widget - dependency - Entity Browser)
  15. Video Embed Field - https://ftp.drupal.org/files/projects/video_embed_field-8.x-1.5.tar.gz
